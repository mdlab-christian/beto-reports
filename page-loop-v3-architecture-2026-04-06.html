page-review
v1.0 (mantida)
Entry point para revisao. Cria/abre arquivo de pendencias manuais.
⚠ Regras Absolutas
R1
Executar cada skill conforme seu SKILL.md. Nao atalhar, nao pular passos internos.
R2
Se skill falha com erro grave, parar e reportar. Nunca continuar no escuro.
R3
P0 bloqueante = resolver ANTES de avancar fase. Nunca arrastar P0.
R4
Dashboard atualizado apos cada skill completar.
R5
page-finalize SEMPRE o ultimo passo. Nunca pular.
R6
NAO abrir HTML no browser durante o loop. Unico open = loop-report no encerramento.
R7
Maximo 2 loops QA. Se apos 2 loops tem P0, documentar e avancar.
R8
Veredicto do senior-review e lei. NAO PRONTA = documentar no loop-report.
