Bugs Conhecidos

P0 — Pendente BUG-04 — Histórico Inconsistente
src/components/mdzap/MdzapConversaTimeline.tsx
MdzapConversaTimeline mostra dados incorretos ou faltantes no histórico de conversas. Possíveis causas: query sem filtro deleted_at, RLS incorreta para o tenant atual, ou divergência de timezone entre banco e frontend. Status: não investigado — prioridade P0 para próxima sessão.
P1 — Excluído do escopo BUG-02 — Document Received (3 nodes)
/MDZAP - Document Received
Workflow simplista — comprovantes recebidos pelo WhatsApp não salvam corretamente em processos_documentos. Necessário expandir de 3 para ~15 nodes: upload para Storage, INSERT Tier 1, sync com Drive. Decisão: tratar separadamente, fora do escopo atual do MdZap.
✅ Corrigido BUG-01 — N8N-REL-CONSULTA Inativo
N8N-REL-CONSULTA (ID: 86F1yHKNlR80MBlR)
Workflow reativado. 3 nós reescritos utilizando this.helpers.httpRequest() para garantir compatibilidade com a versão atual do n8n.
✅ Corrigido BUG-03 — Protestos de Cartório
WH-AUD-01 Auditorias Disparar (ID: BKLW2Egnxd1xCnjL)
Filtro adicionado em WH-AUD-01. Protestos de cartório aparecem no comprovante da auditoria mas não são mais cadastrados como restrições processáveis — comportamento correto conforme regra de negócio.
