E-01 Imagens e documentos não renderizam
P0 Esforço: Médio
▼
"imagem e documentos nao renderizam no front, eu consigo enviar imagens mas nao consigo receber, eu recebo, mas recebo como mensagem expirada."
Causa Raiz
  • Edge Function mdzap-webhook baixa mídia do Meta API — se access_token não está em numero.config, download falha silenciosamente
  • Frontend mostra "Imagem expirada" sem fallback útil
  • Bucket Supabase Storage mdzap-media pode não ter policy READ pública
  • Diferença de formato entre Z-API vs Meta Cloud API (media_url diferente)
Plano Técnico
  1. Verificar bucket mdzap-media → garantir policy pública para READ
  2. Edge Function: logar erro quando access_token ausente, guardar media_id para retry
  3. Frontend: substituir "expirado" por botão "Recarregar" com retry automático
  4. Verificar e normalizar fluxo Z-API vs Meta Cloud API

P0 de Banco

Aplicar ANTES de iniciar qualquer implementação de frontend.

1. whatsapp_templates sem deleted_at
ALTER TABLE public.whatsapp_templates ADD COLUMN IF NOT EXISTS deleted_at TIMESTAMPTZ DEFAULT NULL; CREATE INDEX IF NOT EXISTS idx_whatsapp_templates_org_active ON public.whatsapp_templates(organizacao_id) WHERE deleted_at IS NULL;
2. leads.awaiting_human vs leads.aguardando_humano — sincronizar
-- Trigger de sincronização bidirecional (temporário) CREATE OR REPLACE FUNCTION sync_awaiting_human() RETURNS TRIGGER AS $$ BEGIN IF TG_OP = 'UPDATE' THEN IF NEW.awaiting_human IS DISTINCT FROM OLD.awaiting_human THEN NEW.aguardando_humano := NEW.awaiting_human; ELSIF NEW.aguardando_humano IS DISTINCT FROM OLD.aguardando_humano THEN NEW.awaiting_human := NEW.aguardando_humano; END IF; END IF; RETURN NEW; END; $$ LANGUAGE plpgsql; CREATE TRIGGER sync_awaiting_human_trigger BEFORE UPDATE ON leads FOR EACH ROW EXECUTE FUNCTION sync_awaiting_human();
3. leads.rg — verificar e adicionar se necessário
-- Verificar existência SELECT column_name, data_type FROM information_schema.columns WHERE table_name = 'leads' AND column_name = 'rg'; -- Adicionar se não existir ALTER TABLE public.leads ADD COLUMN IF NOT EXISTS rg TEXT;
