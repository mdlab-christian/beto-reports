Issues P0 — Bloqueadores (Críticos)

1. users — Falta soft delete (deleted_at)

Impacto: Impossível desativar usuário sem DELETE hard. Viola RLS + LGPD.

Ação: Adicionar coluna e índice, atualizar RLS policy.

ALTER TABLE public.users ADD COLUMN IF NOT EXISTS deleted_at TIMESTAMPTZ; CREATE INDEX idx_users_deleted_at ON public.users(deleted_at);

2. organizacoes — Falta soft delete

Impacto: Não é possível suspender org sem perder dados (raiz de toda árvore).

Ação: Adicionar soft delete, atualizar todas queries que lêem organizacoes.

3. RLS users — Policy não valida deleted_at

Impacto: Usuários deletados conseguem logar se tiverem sessão ativa.

Risco: LGPD, segurança.

4. fn_hydrate_user RPC — Não verifica deleted_at

Impacto: Usuário deletado consegue revalidar sessão (reload página).

Localização: migration 20260224011908_c0ff93b8.sql linha 40.

5. TabUsuarios.tsx — Redefinir senha inseguro

Impacto: Função resetPasswordForEmail() chamada sem validações. Risco de enumeração de emails.

Solução: Criar RPC segura fn_reset_user_password com SECURITY DEFINER.

Issues P1 — Importantes (Ação Recomendada)

users.setor_id — FK sem validação de org

Foreign key não valida que setor pertence à mesma org. Violação de isolamento tenant.

advogados.ativo — Sem índice

Tabela sem índice para coluna ativo. Queries lentas com 100+ advogados.

config_* tabelas — Sem RPC de validação

Atualizadas direto via REST. Dados inconsistentes (ex: etiquetas órfãs).

users.config_ui — JSONB sem validação

Coluna não valida schema. JSON corrompido pode quebrar login.

organizacao_setores — Nome não é único

Dois setores com mesmo nome permitidos. Ambigüidade em filtros.

advogados.oab — Sem validação de formato

Valores inválidos aceitos. Afeta buscas e filtros por OAB.

RLS advogados — Policy vaga

Sem granularidade por setor. Gestor de setor vê todos advogados da org.

usuarios_organizacoes — Tabela fantasma?

Mencionada em tipos TS mas não existe. Usuários usam organizacao_id direto em users.

SQL de Correção P0 — Pronto para Produção
