Problemas Identificados

P1 — BLOCKER
wa_numero_id hardcoded no FACEBOOK-INBOUND
O flow k9377WTyVJYOnD4e usa um ID fixo para o número WhatsApp em vez de fazer lookup dinâmico. Isso impede que a arquitetura multi-SDR funcione, pois não é possível rotear mensagens de múltiplos números para diferentes agentes.
Fix F1: lookup em whatsapp_numeros WHERE wa_instance_id = phone_number_id recebido
P2 — BLOCKER
FACEBOOK-INBOUND não chama o Orchestrator
Após salvar a mensagem, o flow não faz nenhuma chamada ao Orchestrator Julia. Isso significa que Julia nunca responde a nenhuma mensagem recebida — o loop de conversa IA está quebrado.
Fix F1: adicionar step Router que lê agente_slug e chama o webhook correspondente
P3 — BLOCKER
Orchestrator hardcoda slug sdr_leads
O flow HLdJDx6D45EcER9Z busca sempre o módulo sdr_leads em mdzap_ai_modules, ignorando o campo agente_slug passado pelo Router. SDR Docs e SDR Acordos nunca carregariam seu system_prompt.
Fix F1: receber agente_slug no body do POST e usar dinamicamente no SELECT mdzap_ai_modules
P4 — BUG
API key Anthropic hardcoded no Orchestrator
A chave da Anthropic está embutida diretamente no node HTTP do Orchestrator, em vez de ser lida via variável de ambiente. Representa risco de segurança e impossibilita rotação de chave centralizada.
Fix F1: substituir por $env.ANTHROPIC_API_KEY nas credenciais do node

Insight do Framework Legado — Message Grouper

O que era o W1 (framework .mmd)

O framework antigo implementava um Message Grouper com delay de 10 segundos antes de chamar a IA. Ele agrupava mensagens enviadas em rápida sucessão para evitar respostas fragmentadas.

Usuário: "oi" ─┐ Usuário: "sou o João" ├─ delay 10s → 1 chamada IA Usuário: "tenho uma dívida" ─┘
Recomendação para F2

Implementar no FACEBOOK-INBOUND antes de chamar o Router:

1
Receber mensagem
Save imediato em whatsapp_mensagens
2
Wait Node (10s)
Aguarda possíveis mensagens seguintes
3
Check: nova mensagem chegou?
Se sim → reinicia timer. Se não → chama Router
4
Router SDR
Todas as mensagens agrupadas em 1 contexto
