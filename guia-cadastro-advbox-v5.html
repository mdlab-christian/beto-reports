REGRA #1 — Grupo de Acao CRITICO

O campo groups_id (Grupo de Acao) NAO PODE ser alterado via API apos a criacao do processo.
A API retorna "Dados atualizados com sucesso" mas IGNORA o valor silenciosamente. Se errar, a unica correcao e MANUAL na interface do AdvBox. Portanto: ACERTAR NA PRIMEIRA VEZ E OBRIGATORIO.

Por que o Grupo de Acao importa?

  • Vincula o processo ao advogado responsavel (OAB correta)
  • Puxa automaticamente o Tipo de Acao correto (ex: SP535101 → tipo "SAO PAULO")
  • Determina em qual OAB as intimacoes serao recebidas
  • Errar = processo vinculado ao advogado/tribunal errado = retrabalho manual

Logica de Mapeamento ALGORITMO

Arvore de Decisao

O mapeamento depende de DOIS fatores: (1) Tribunal do CNJ e (2) Advogado responsavel.

1. Extrair tribunal do numero CNJ:
├─
CNJ contem .8.21. → Tribunal = TJRS
├─
CNJ contem .8.24. → Tribunal = TJSC
├─
CNJ contem .8.26. → Tribunal = TJSP
├─
CNJ contem .4.04. → Tribunal = TRF4 (tratar como SC)
└─
Outro padrao → PARAR e pedir orientacao
2. Determinar advogado (pela planilha ou instrucao):
├─
Advogado = Christian + TJRS → groups_id = 225949
├─
Advogado = Christian + TJSC/TRF4 → groups_id = 498042
├─
Advogado = Christian + TJSP → groups_id = 498044
├─
Advogado = Heloisa + TJRS → groups_id = 498045
└─
Advogado = Heloisa + TJSC → groups_id = 498046

def get_groups_id(cnj: str, advogado: str = 'christian') -> str:
    """
    Retorna o groups_id correto baseado no TRIBUNAL do CNJ
    e no advogado responsavel.

    REGRA CRITICA: Mapear pelo tribunal do CNJ, NUNCA pela OAB da planilha.
    O groups_id NAO PODE ser alterado apos criacao.
    """
    cnj = str(cnj).strip()
    advogado = advogado.strip().lower()

    # 1. Detectar tribunal pelo padrao CNJ
    if '.8.21.' in cnj:
        tribunal = 'RS'
    elif '.8.24.' in cnj:
        tribunal = 'SC'
    elif '.8.26.' in cnj:
        tribunal = 'SP'
    elif '.4.04.' in cnj:
        tribunal = 'SC'   # TRF4 → tratar como SC
    else:
        raise ValueError(f'Tribunal desconhecido no CNJ: {cnj}')

    # 2. Mapear advogado + tribunal → groups_id
    MAPA = {
        ('christian', 'RS'): '225949',   # CHRISTIAN RS116571
        ('christian', 'SC'): '498042',   # CHRISTIAN SC74025
        ('christian', 'SP'): '498044',   # CHRISTIAN SP535101
        ('heloisa',   'RS'): '498045',   # HELOISA RS131163
        ('heloisa',   'SC'): '498046',   # HELOISA SC75466
    }

    # Normalizar nome do advogado
    if 'helo' in advogado or '131163' in advogado or '75466' in advogado:
        adv_key = 'heloisa'
    else:
        adv_key = 'christian'

    key = (adv_key, tribunal)
    if key not in MAPA:
        raise ValueError(f'Combinacao sem mapeamento: {adv_key} + {tribunal}')

    return MAPA[key]
Como determinar o advogado?
- Se a planilha tem coluna "Advogado" ou "OAB": usar o NOME do advogado (Christian ou Heloisa), NAO o numero da OAB para determinar o tribunal
- Se a planilha so tem OAB: RS116571/SC74025/SP535101 = Christian; RS131163/SC75466 = Heloisa. Mas o TRIBUNAL vem do CNJ
- Na duvida: se OAB contem "131163" ou "75466" → Heloisa, caso contrario → Christian

A busca do AdvBox (POST /search) retorna matches parciais ordenados por relevancia. Exemplo:

  • Buscar "VERO" retorna "Adilson Fernando Severo" antes de "Vero S.A."
  • Solucao: buscar com mais palavras (ex: "VERO S.A") ou validar o resultado
def search_customer(session, name):
    """Busca cliente — usar primeiras 3 palavras para match"""
    words = name.strip().split()
    query = ' '.join(words[:3]) if len(words) >= 3 else name
    results = session.post(BASE_URL + '/search', data={'search': query}).json()
    customers = results.get('customers', [])
    if customers:
        # Verificar se o match faz sentido
        found_name = customers[0].get('filter', '')
        if similar_enough(found_name, name):
            return extract_id(customers[0])
    return None

"""
AdvBox Automation v5 — Template para cadastramento correto
groups_id mapeado por TRIBUNAL do CNJ (regra obrigatoria)
"""
import json, time, re, urllib.parse
from datetime import datetime
import openpyxl, requests

# ============================================================
# CONFIG
# ============================================================
STATE_FILE      = '/tmp/advbox_auth.json'
BASE_URL        = 'https://app.advbox.com.br'
SLEEP_BETWEEN   = 1.5

# Constantes do AdvBox
DEFAULT_TYPE_LAWSUIT = '1880677'
DEFAULT_STEPS_ID     = '3'
DEFAULT_STAGES_ID    = '789277'
DEFAULT_USERS_ID     = '140216'
CUSTOMERS_ORIGIN     = '302582'  # INTERNET

# ============================================================
# MAPEAMENTO POR TRIBUNAL (REGRA CRITICA)
# ============================================================
GROUPS_MAP = {
    # (advogado, tribunal) → groups_id
    ('christian', 'RS'): '225949',   # CHRISTIAN RS116571
    ('christian', 'SC'): '498042',   # CHRISTIAN SC74025
    ('christian', 'SP'): '498044',   # CHRISTIAN SP535101
    ('heloisa',   'RS'): '498045',   # HELOISA RS131163
    ('heloisa',   'SC'): '498046',   # HELOISA SC75466
}

def extrair_tribunal(cnj: str) -> str:
    """Extrai tribunal do numero CNJ."""
    if '.8.21.' in cnj: return 'RS'   # TJRS
    if '.8.24.' in cnj: return 'SC'   # TJSC
    if '.8.26.' in cnj: return 'SP'   # TJSP
    if '.4.04.' in cnj: return 'SC'   # TRF4 → SC
    raise ValueError(f'Tribunal desconhecido: {cnj}')

def identificar_advogado(oab: str, advogado: str = '') -> str:
    """Identifica se e Christian ou Heloisa."""
    texto = f'{oab} {advogado}'.upper()
    if '131163' in texto or '75466' in texto or 'HELO' in texto:
        return 'heloisa'
    return 'christian'

def get_groups_id(cnj: str, oab: str = '', advogado: str = '') -> str:
    """
    Retorna groups_id CORRETO baseado no tribunal do CNJ.
    NUNCA mapear pela OAB da planilha diretamente.
    """
    tribunal = extrair_tribunal(cnj)
    adv = identificar_advogado(oab, advogado)
    key = (adv, tribunal)
    if key not in GROUPS_MAP:
        raise ValueError(f'Sem mapeamento: {adv} + {tribunal}')
    return GROUPS_MAP[key]

# ============================================================
def build_session():
    with open(STATE_FILE) as f:
        cookies_list = json.load(f)
    session = requests.Session()
    session.headers.update({
        'User-Agent': 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7)',
        'X-Requested-With': 'XMLHttpRequest',
        'Accept': 'application/json',
        'Referer': f'{BASE_URL}/processos',
    })
    for c in cookies_list:
        session.cookies.set(c['name'], c['value'],
                           domain=c.get('domain', 'app.advbox.com.br'))
        if c['name'] == 'XSRF-TOKEN':
            session.headers['X-XSRF-TOKEN'] = urllib.parse.unquote(c['value'])
    return session

def get_csrf(session):
    r = session.get(f'{BASE_URL}/processos',
                    headers={'Accept': 'text/html'})
    m = re.search(r'csrf-token" content="([^"]+)"', r.text)
    return m.group(1) if m else None

# ADVBOX API
# ============================================================
def search(session, query):
    r = session.post(f'{BASE_URL}/search', data={'search': query})
    r.raise_for_status()
    return r.json()

def find_lawsuit(session, cnj):
    data = search(session, cnj.split('-')[0])
    for l in data.get('lawsuits', []):
        if cnj in l.get('number', ''):
            m = re.search(r'lawsuits/(\d+)', l.get('link', ''))
            return m.group(1) if m else None
    return None

def find_or_create_customer(session, csrf, name, cache):
    key = name.strip().upper()
    if key in cache:
        return cache[key]

    words = name.strip().split()
    query = ' '.join(words[:3]) if len(words) >= 3 else name
    data = search(session, query)

    for c in data.get('customers', []):
        m = re.search(r'customers/(\d+)', c.get('link', ''))
        if m:
            cache[key] = m.group(1)
            return m.group(1)

    # Criar
    time.sleep(SLEEP_BETWEEN)
    tipo = '2' if is_pj(name) else '1'
    r = session.post(f'{BASE_URL}/customers', data={
        '_token': csrf, 'name': name, 'type': tipo,
        'customers_origins_id': CUSTOMERS_ORIGIN,
        'notes': 'Criado via automacao',
    })

    try:
        resp = r.json()
        link = resp.get('url', '') or resp.get('redirect', '')
        for pat in [r'customers/(\d+)', r'\?id=(\d+)']:
            m = re.search(pat, link)
            if m:
                cache[key] = m.group(1)
                return m.group(1)
    except: pass
    return None

PJ_KEYWORDS = ['S.A', 'LTDA', 'S/A', 'EIRELI', 'BANCO', 'FUNDO',

def is_pj(name):
    return any(k in name.upper() for k in PJ_KEYWORDS)

def create_lawsuit(session, csrf, cnj, cust_id, contr_id,
                   cliente, contraria, groups_id):
    data = [
        ('_token', csrf),
        ('customer', ''),
        ('customer_type[]', '1'),
        ('customer_type[]', '3'),
        ('customer_list[]', cust_id),
        ('contrary_list[]', contr_id),
        ('notes', f'Parte Autora: {cliente} | Contraria: {contraria}'),
        ('users_id', DEFAULT_USERS_ID),
        ('groups_id', groups_id),
        ('type_lawsuits_id', DEFAULT_TYPE_LAWSUIT),
        ('steps_id', DEFAULT_STEPS_ID),
        ('stages_id', DEFAULT_STAGES_ID),
        ('process_number', cnj),
    ]
    # Campos opcionais
    for f in ['year','judiciary','county','stick','courts_id',
              'systems_id','protocol_number','originating_process',
              'folder','date','fees_expec','fees_money',
              'fees_percentage','contingency']:
        data.append((f, ''))

    r = session.post(f'{BASE_URL}/lawsuits', data=data)
    try:
        resp = r.json()
        m = re.search(r'lawsuits/(\d+)', resp.get('url', ''))
        return m.group(1) if m else None, resp.get('message', '')
    except:
        return None, f'HTTP {r.status_code}'

def verify_group(session, lawsuit_id, expected_group):
    """Verifica se o grupo foi aplicado corretamente."""
    r = session.get(f'{BASE_URL}/lawsuits/{lawsuit_id}/edit',
                    headers={'Accept': 'text/html'})
    m = re.search(r'name="groups_id".*?value="(\d+)"[^>]*selected',
                  r.text, re.S)
    actual = m.group(1) if m else '?'
    return actual == expected_group, actual

# ============================================================
# MAIN — ADAPTAR PARA CADA PLANILHA
# ============================================================
def main(excel_file, header_row=1, data_start_row=2,
         col_cnj=0, col_cliente=1, col_contraria=2,
         col_oab=None, col_advogado=None):
    """
    Parametros adaptaveis por planilha:
    - header_row: linha dos headers
    - data_start_row: primeira linha de dados
    - col_cnj, col_cliente, col_contraria: indices das colunas
    - col_oab: indice da coluna OAB (para identificar advogado)
    - col_advogado: indice da coluna advogado
    """
    session = build_session()
    csrf = get_csrf(session)
    cache = {}

    wb = openpyxl.load_workbook(excel_file)
    ws = wb.active

    rows = []
    for row in ws.iter_rows(min_row=data_start_row,
                            max_row=ws.max_row, values_only=True):
        if row[col_cnj] and str(row[col_cnj]).strip():
            cnj = str(row[col_cnj]).strip()
            cliente = str(row[col_cliente]).strip()
            contraria = str(row[col_contraria]).strip()
            oab = str(row[col_oab]).strip() if col_oab else ''
            advogado = str(row[col_advogado]).strip() if col_advogado else ''

            groups_id = get_groups_id(cnj, oab, advogado)
            rows.append({
                'cnj': cnj, 'cliente': cliente, 'contraria': contraria,
                'groups_id': groups_id, 'oab': oab, 'advogado': advogado,
            })

    # PRE-VALIDACAO
    print(f'Total: {len(rows)} processos')
    by_group = {}
    for r in rows:
        by_group.setdefault(r['groups_id'], []).append(r)
    for gid, procs in sorted(by_group.items()):
        print(f'  Grupo {gid}: {len(procs)} processos')

    print('\nConfirmar? (Ctrl+C para cancelar)')
    time.sleep(3)

    # EXECUCAO
    for i, proc in enumerate(rows, 1):
        cnj = proc['cnj']
        groups_id = proc['groups_id']

        print(f'[{i:3d}/{len(rows)}] {cnj} → grupo {groups_id}')
        time.sleep(SLEEP_BETWEEN)

        # Verificar duplicata
        existing = find_lawsuit(session, cnj)
        if existing:
            print(f'  → Ja existe (ID {existing}), pulando')
            continue

        # Buscar/criar partes
        time.sleep(SLEEP_BETWEEN)
        cid = find_or_create_customer(session, csrf, proc['cliente'], cache)
        time.sleep(SLEEP_BETWEEN)
        kid = find_or_create_customer(session, csrf, proc['contraria'], cache)

        if not cid or not kid:
            print(f'  ✗ Partes nao encontradas')
            continue

        # Criar processo
        time.sleep(SLEEP_BETWEEN)
        lid, msg = create_lawsuit(session, csrf, cnj, cid, kid,
                                  proc['cliente'], proc['contraria'],
                                  groups_id)

        if lid:
            # VERIFICAR grupo aplicado
            time.sleep(SLEEP_BETWEEN)
            ok, actual = verify_group(session, lid, groups_id)
            if ok:
                print(f'  ✓ ID {lid} — grupo {groups_id} VERIFICADO')
            else:
                print(f'  ⚠ ID {lid} — grupo ERRADO! esperado={groups_id} atual={actual}')
        else:
            print(f'  ✗ Erro: {msg}')
