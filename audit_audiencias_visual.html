⚠️ Issues Encontrados

P1 Índices de Performance Faltando
Não foi possível confirmar a existência dos índices esperados via Supabase SDK. Queries com filtro WHERE organizacao_id = ? AND deleted_at IS NULL executarão em O(n) sem índice.
Performance crítica em /audiencias com 5k+ registros. Latência aumentará drasticamente.
CREATE INDEX IF NOT EXISTS idx_audiencias_org_id ON public.audiencias(organizacao_id); CREATE INDEX IF NOT EXISTS idx_audiencias_org_id_active ON public.audiencias(organizacao_id) WHERE deleted_at IS NULL;
P2 Trigger updated_at Não Confirmado
Não foi possível verificar se trigger set_updated_at está presente na tabela via SDK. Requer acesso SQL direto ao information_schema.triggers.
Se ausente, updated_at não será atualizado automaticamente.
-- Verificar: SELECT trigger_name FROM information_schema.triggers WHERE event_object_table = 'audiencias' AND trigger_name ILIKE '%updated%'; -- Criar se não existir: CREATE TRIGGER set_updated_at BEFORE UPDATE ON public.audiencias FOR EACH ROW EXECUTE FUNCTION public.update_updated_at_column();
